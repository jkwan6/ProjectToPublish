Website Structure:

* Website is still in development.

  • Website is an SPA Website, communicating with a Rest API.
  • Website built via Typescript and C#.
  • Front end handled via Angular
  • Front end stylized via Angular Material
  • Front End 3D Assets handled via Three Js
  • Front End Physics handled via Rapier Js
  • Back end handled via ASP.Net Core
  • Authentication Handled via ASP.Net Identity.
  • Authentication Handled via JWT - Access Token, Refresh Tokens and AppSessions.
  • Database Handled via EF Core layered over MSSQL
